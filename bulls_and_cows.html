BULLS AND COWS Bulls and cows is a classic code-breaking game. The AI has a secret 4-digit number. Each round, the user is given the number of bulls (right number right location) and cows (correct number in wrong location). For example, if the answer is 1234 and the user guesses 4321, they would see 4 cows. A guess of 1289 would result in 2 bulls and 0 cows. My responsbilities: UX/UI design, typography, abstract drawing of bulls and cows, original game logic and AI programming (Python)
Three Modes
Solo: The user guesses the number from the AI. VS: The user and AI take turns guessing each other's number to see who wins first. AI: It's the AI's turn to guess the user's number.
